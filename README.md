# cdiscount price parser

## Installation
```python
pip install cdiscountpriceparser
```

## Usage
```python
python
from price_parser import parse_price

# Return "False"
parse_price()

# Return "1776.6"
parse_price("del5397184246030")
```

## Development
```python
pip install -e .[dev]
```

## Docker usage
```
docker build --pull --rm -f "Dockerfile" -t cdiscount:latest "."
```
